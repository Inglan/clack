## Features

* Browse trending videos
* View Video stats
	* Like count
	* Comment Count
	* Share Count
* View a user's profile
	* Profile picture
	* User Stats
		* Following count
		* Followers count
		* Hearts count
	* Signature
	* User's videos
	* User's liked videos (if public)
* Share a user
* Share a video
